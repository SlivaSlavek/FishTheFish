# Fish The Fish 🇬🇧
Made by Slávek Slíva, *Czech republic, 2026*
## About game
Fish The Fish is small game about fishing. You catch various fishes and collect them into you Fish index. You also get score by catching fish, rarer fishes have greater value. 
Game was made by one developer and programmed in Java.
## Basics of how does the game work
- Game UI was made using Java swing.
- Timings of fish spawn, despawns etc. were made with Java Swing Timers.
## How to play
- Instructions are ingame in "HELP" section.
- To play, you can just use folder FishTheFishGameJar which contains .jar file of Game, Icon, saving file and shortcut. Everything has to stay in same folder, expect the shortcut of .jar, you can put for example on your desktop.





# Fish The Fish 🇨🇿
Vytvořil Slávek Slíva, *Česká republika, 2026*
## O hře
Fish The Fish je malá hra o rybaření. Chytáte různé ryby a sbíráte je do indexu ryb. Dostáváte také body za chytání ryb, čím vzácnější ryba, tím vyšší má hodnotu. Hra je v angličtině.
Hra byla vytvořena jedním vývojářem a byla naprogramována v Javě.
## Základy z toho jak hra funguje
- UI hry bylo uděláno s využitím Java swingu
- časovače spawnování ryb, despawnování atd. byly vytvořeny Java Swing Timery.
## Jak hrát
- Návod je přímo ve hře v sekci "HELP"
- Pro hraní Vám stačí složka FIshTheFishGameJar, která obsahuje .jar soubor, ikonku, ukládací soubor a zástupce programu. Všechno musí zůstat ve stejné složce, až na zástupce programu .jar, kterého si můžete hodit např. na plochu.
